## Purpose

This document defines the strategic database architecture for the AIG Platform and the North Star ecosystem.

## Principles

- Relational data model for core transactional systems
- Strong consistency for critical business processes
- Clear ownership of domain data
- Secure access controls
- Backup and recovery by design
- Schema evolution through versioned migrations
